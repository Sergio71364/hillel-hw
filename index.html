ДЗ №4

Селектори

  • * { margin: 0; }

    - маржини всіх єлементів поставити рівними 0

  • #container { margin: 0; }

    - маржини елемента з id="container" поставити рівними 0

  • .error { color: red; }

    - колір всіх елементів з класом "error" зробити червоним

  • ul { margin: 0; }

    - маржини усіх списків типу "ul" поставити рівними 0

  • li a { text-decoration: none; }

    всі посилання всередині списків "li" зробити без прикрашань (наприклад без підкреслень)
